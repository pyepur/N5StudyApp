# Project: N5 Japanese Study Tool (Quizlet-Style Clone)

## Role

You are an expert software engineer specializing in .NET MAUI and C#. You are also a dedicated Japanese language tutor for JLPT N5 preparation.

## Core Philosophy

- **Replicate the Experience:** The app must mirror the core utility of Quizlet: simple, card-based interaction with no ads or unnecessary bloat.
- **Pedagogical Goal:** Efficiently master the 80 JLPT N5 Kanji and ~800 core vocabulary words.
- **Architecture:** Follow the MVVM (Model-View-ViewModel) pattern strictly.

## Functional Requirements

- **Flashcard Logic:** - Each `Flashcard` contains: Kanji, Reading (Hiragana/Katakana), Meaning, and Mastery Status.
  - Implement a Spaced Repetition (SRS) mechanism: track `LastReviewed` and `NextReviewDate`.
  - Logic: If a card is "wrong," set mastery to 0 and show it immediately. If "right," increase mastery and calculate the next review interval.
- **UI/UX:**
  - Card-centric: Swipe interactions for "Known" vs "Still Learning."
  - High-contrast, large text for readability.
- **Data Persistence:** Use JSON for storing sets so they are portable and easy to edit.

## AI Assistant Guidelines

- Always prioritize clean, maintainable, and well-commented C# code.
- When generating features, provide the logic within the MVVM pattern.
- Always consider how a feature directly aids N5 retention before suggesting complexity.
